Display beautifully clean, customizable, and responsive feeds with Thumbnails, Masonry, Instagram, Blog, Fancy Gallery and Grid Rotator Layout

== Description ==

Display Instagram images from any non-private Instagram accounts. There is multiple options to display layout, you can display Thumbnail, Masonry, Instagram, Blog , Fancy and Grid Rotator with other multiple options like shorting, display number of images, number of columns and others options too.

for demo visite : [Plugin Demo](http://demo.spiderbuzz.com/plugins/instagram-buzz-feed/)


= Features =
* Super **easy to set up**
* Display photos from **Instagram accounts**
* Completely **responsive** and mobile friendly - layout looks great on any screen size and in any container width
* **Completely customizable** - Customize layout(Thumbnail, Masonry, Instagram, Blog, Fancy Gallery, Grid Rotator), 
* **Feed types** Select feed types to display Instagram feeds from different options (My Feed, My Recent Media, Likes, Popular Feed, From any user, Popular tags)

* Option to show likes, comments and descriptions

== Installation ==
= First Method =
1. Install the Instagram plugin either via the WordPress plugin directory, or by uploading the files to your web server (in the `/wp-content/plugins/` directory).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Instagram Feed' settings page to obtain your Instagram Access Token and User ID and configure your settings.
4. Use the shortcode `[buzz_instagram_feed]` in your page, post or widget to display your photos.
5. You can use widget to display feeds

= Second Method =
**Perform a new installation**
*After downloading the ZIP file of the Instagram Feed WD plugin,

* Log in to the administrator panel.
* Go to Plugins Add > New > Upload.
* Click "Choose file" ("Browse") and select the downloaded zip file of the Instagram Feed plugin.
* **For Mac Users**

*Go to your Downloads folder and locate the folder with the Instagram Feed WD. Right-click on the folder and select Compress. Now you have a newly created .zip file which can be installed as described here.
* Click "Install Now" button.
* Click "Activate Plugin" button for activating the Instagram Feed WD.
* If the installation does not succeed, please contact us at info@spiderbuzz.com.
